# Bibliography Reference

Add entries to `refs.bib`:

```bibtex
@book{key,
  author = {Прізвище, І. І.},
  title = {Назва книги},
  publisher = {Видавництво},
  year = {2024},
}

@article{key,
  author = {Author, A.},
  title = {Article Title},
  journal = {Journal Name},
  year = {2024},
  volume = {1},
  pages = {1--10},
  doi = {10.1000/example},
}

@misc{key,
  author = {{Organization}},
  title = {Page Title},
  year = {2024},
  howpublished = {\url{https://example.com}},
  note = {Дата звернення: грудень 2024},
}
```

Cite: `\cite{key}` or `\cite{key1, key2}`
